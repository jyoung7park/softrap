Softrap 소개

Softrap은 일상에서 재미와 편의를 더해주는 AI 기반 웹 서비스를 만들어가고 있습니다. 복잡한 설치 없이 브라우저 하나로 다양한 도구와 테스트를 무료로 이용할 수 있습니다.

🎯 서비스 소개

🎱

로또 번호 추첨

1부터 45번 중 무작위로 6개의 메인 번호와 보너스 번호를 즉시 추첨합니다. 색상별로 구분된 번호 볼과 애니메이션으로 실제 추첨 느낌을 전달합니다.

🐾

동물상 테스트

AI가 사진을 분석해 개상인지 고양이상인지 알려줍니다. Google Teachable Machine 기반 이미지 분류 모델을 사용하며, 업로드된 사진은 서버에 저장되지 않습니다.

🔒 개인정보 보호

Softrap은 이용자의 개인정보 보호를 최우선으로 생각합니다. 동물상 테스트에서 업로드되는 사진은 사용자의 브라우저 내에서만 처리되며 외부 서버로 전송되지 않습니다. 제휴 문의를 통해 수집된 정보는 문의 처리 목적으로만 사용됩니다.

자세한 내용은 개인정보처리방침을 참고해 주세요.

💡 기술 스택

Softrap은 순수 HTML, CSS, JavaScript로 제작되어 빠른 속도와 가벼운 용량을 자랑합니다. 별도의 서버 없이 정적 사이트로 운영되어 높은 안정성을 제공합니다.

  • 프론트엔드: HTML5, CSS3, Vanilla JavaScript
  • AI 모델: Google Teachable Machine (TensorFlow.js)
  • 배포: Cloudflare Pages
  • 댓글: Disqus
  • 문의 폼: Formspree
  • 광고: Google AdSense
